Avatar

Make Responsive Avatars with one8Ui. Various templates like a circle avatar, square and letter avatar. Avatars are used widely across the Website for profiles, blogs and many more.

Profile Avatar

Profile Avatars can be created using the class avatar, inside the <img> tag.

Sizes Avatar

Sizes Avatars are available in 3 sizes small,medium and large.You need to include class avatar and for size add class according to size sm | md | lg.

Square Avatar

Adding a class square to existing class, will generate a square shaped avatar.
